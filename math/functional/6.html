本章总假定 `X, Y` 等是 Banach 空间.

紧算子

基本性质

    紧算子 设 `A` 是两个 Banach 空间 `X, Y` 间的线性算子, 则以下几条等价:
  1. `X` 中单位球的像的闭包 `bar(A(B_1))` 在 `Y` 中紧 (取闭包是因为单位球的像可能不是闭集).
  2. `X` 中任意有界集的像的闭包在 `Y` 中紧.
  3. `X` 中任意有界点列 `{x_n}` 的像 `{A x_n}` 有收敛子列.
    4. 若 `A` 满足任意一条则称 `A` 是紧线性算子, 简称紧算子, 记作 `A in fr C(X, Y)`. 简记 `fr C(X, X) = fr C(X)`.
  1. 有限维线性变换都是紧的.
  2. 无穷维空间中 `I` 不是紧算子, 因为单位球不紧.
    紧算子的性质
  1. `fr C(X, Y)` 是 `cc L(X, Y)` 的闭子空间.
  2. 紧算子 `A` 在 `X` 的闭子空间 `X_0` 上的限制 `A|_(X_0)` 仍是紧算子.
  3. 紧算子的像 `"Im" A` 可分.
  4. 连续算子与紧算子的复合是紧算子.

全连续线性算子 设 `X, Y` 是 Banach 空间. 称 `A in cc L(X, Y)` 是全连续的, 如果对任意弱收敛点列 `x_n ⇀ x` 可以推出强收敛 `A x_n to A x`.

紧算子是全连续的; 自反空间 `X` 上的全连续线性算子是紧的.

紧算子的共轭算子是紧的.

用有穷秩算子逼近紧算子

  1. 设 `T in cc L(X,Y)`, 如果 `"rank" T` `:= dim "Im" T lt oo`, 则称它是有穷秩算子, 记为 `T in F(X, Y)`.
  2. 设 `y in Y`, `f in X^ast`, 定义它们的张量积 `y ox f` 如下: `y ox f: X to Y`
    `(y ox f)(x) = (:f, x:) y`.     想像 `x, y` 是列向量, `f` 是行向量, 则 `(y f) x = y (f x)`, 其中 `y f` 是一个秩 `le 1` 的矩阵. 因此我们也称 `y ox f` 为秩 1 算子.

设 `"rank" T = n`, 则它可以写为 `n` 个秩 1 算子之和 `T = sum_(i=1)^n y_i ox f_i`.

有限维空间中的有界闭集是紧集, 故 `F(X, Y) sube fr C(X, Y)`. 但 `F(X, Y)` 是否在 `fr C(X, Y)` 中稠密? 我们将要证明: 在 Hilbert 空间这是对的; 若 Banach 空间上存在 Schauder 基, 这也是对的.

Hilbert 空间上 `bar(F(X)) = fr C(X)`.

Schauder 基 是指 Banach 空间 `X` 的一列元素 `{e_n}`, 且存在一列泛函 `C_n: X to K` 使得 `x = sum_(n ge 1) C_n(x) e_n`, `quad AA x in X`. 换言之 `x` 可以在基 `{e_n}` 上展开为级数.

若 `X` 有 Schauder 基, 则 `X` 可分. 事实上 `{e_n}` 的有理系数的线性组合在 `X` 中稠密.

在 Schauder 基定义中, 每一个 `C_n` 都是连续线性泛函.

若 Banach 空间上存在 Schauder 基, 则 `bar(F(X)) = fr C(X)`.

算子方程的可解性条件 (Riesz-Fredholm 理论)

本节的主题是算子方程 `T x = y` 的可解性条件, 其中 `x, y in X`, `A = I - T in fr C(X)`.

Fredholm 结论

    线性代数方程的 Fredholm 结论 令 `X = RR^n`, `T^ast` 代表 `T` 的转置. 在线性代数中, 我们已有如下结论:
  1. 齐次方程 `T x = 0` 和共轭齐次方程 `T^ast x = 0` 的解空间维度相等: `dim "Ker" T = dim "Ker" T^ast`.
  2. 特别当 `T x = 0` 只有零解时, `T^ast x = 0` 也只有零解. 此时对任意 `y in X`, 原方程 `T x = y` 都有唯一解.
  3. 如果 `T x = 0` 有非零解, 则 `T^ast x = 0` 也有非零解. 此时原方程 `T x = y` 有解当且仅当对 `AA z in "Ker" T^ast` 成立 `z _|_ y`, 换言之 `(:z, y:) = 0`.

对 3. 进行说明: `T x = y` 有解当且仅当 `y` 可以写为 `T` 的列向量 `T_j` 的线性组合, 即 `y in "span"(T_1, cdots, T_n)`. 对任意 `z in "Ker"T^ast`, 有 `z _|_ T_j`, `j = 1, cdots, n`. 于是 `y in "span"(T_1, cdots, T_n)` `iff (AA z in "Ker" T^ast)` `z _|_ y`.

我们将上述结论推广到一般的算子方程. 记 `"Im" T := T(X)`, `quad "Ker"T := {x in X: T x = 0}`. 规定 `f _|_ x iff (:f, x:) = 0`. 又对 `M sube X`, `F sube X^ast` 定义 (谁垂直 `M`) `quad {::}^_|_M := {f in X^ast: (AA x in M) f _|_ x}`,
(`N` 垂直谁) `quad N^_|_ := {x in X: (AA f in N) f _|_ x}`. 易知 `M sube ({::}^_|_M)^_|_`, `N sube {::}^_|_(N^_|_)`. 最后, 对闭线性子空间 `M sube X`, 定义它的余维数 `op(codim) M := dim(X//M)`.
运用这些记号叙述结论如下:

    Fredholm 结论 设 `A in fr C(X)`, `T = I - A`, 则
  1. `"Ker"T`, `"Im"T`, `"Ker"T^ast`, `"Im"T^ast` 均为闭线性子空间.
  2. `"Im"T = ("Ker"T^ast)^_|_`, `"Im"T^ast = {::}^_|_("Ker"T)`.
  3. `"Ker"T = 0` `rArr "Im"T = X`, 换言之 `T` 是单射 `rArr T` 是满射.
  4. `dim "Ker"T` `= dim "Ker"T^ast` `= op(codim)"Im"T` `= op(codim)"Im"T^ast` `lt oo`.

结论 3. 可以看作 4. 的特殊情形, 这是因为 `"Ker"T = 0` 意味着 `"Ker"T^ast = 0`, 从而由 2. 知道 `"Im"T = ("Ker"T^ast)^_|_ = X`.

结论的证明

Fredholm 结论一 若 `A in fr C(X)`, 则 `T = I - A` 是闭值域算子, 即 `"Im"T = bar("Im"T)`.
此时由于 `A^ast` 是紧算子, `T^ast = I - A^ast` 也是闭值域的.

  1. `"Ker"T` 是闭的: `AA x in bar("Ker" T)`, `AA epsi gt 0`, `EE x_epsi in "Ker"T` 使得 `||x - x_epsi|| lt epsi`. 于是 `||T x|| = ||T x - T x_epsi|| lt ||T|| epsi`. 由 `epsi` 的任意性知 `T x = 0`, 于是 `x in "Ker"T`.
  2. 我们可以对闭子空间 `"Ker"T` 作商, 得到商空间 `X//"Ker"T`. 商空间中的一般元素是 `tilde x`, 它表示 `x` 所在的等价类 `"Ker"T + x`. 商空间的范数是将 `"Ker"T` 中的元素视为 0, 其它方面与 `X` 的范数相同, 因此 `||tilde x|| le ||x||`.
  3. 考察 `T` 在商空间诱导的有界线性算子 `tilde T: X//"Ker"T to X`,
    `tilde T tilde x := T x`.     显然 `"Im"tilde T = "Im"T`, `"Ker"tilde T = tilde 0`, 从而 `tilde T` 是单射.
  4. 为证明 `"Im"T = "Im"tilde T` 闭, 只需证 `tilde T^-1` 连续. 如若不然 `tilde T^-1` 无界, 存在 `{tilde y_n}` 和正数列 `M_n to oo` 使得 `||tilde T^-1 tilde y_n|| ge M_n ||tilde y_n||`, 令 `tilde x_n = tilde T^-1 tilde y_n` 得到 `||tilde x_n|| ge M_n ||tilde T tilde x_n||`. 不妨设 `||tilde x_n|| = 1`, 于是 `tilde T tilde x_n to 0`, 从每个等价类 `tilde x_n` 中选取代表元 `x_n`, 于是存在有界点列 `{x_n}` 使得 `T x_n to 0`.
  5. 另一方面由 `A` 紧, 有子列 `{x_(n_k)}` 使得 `A x_(n_k) to z`, 从而 `x_(n_k) = A x_(n_k) + T x_(n_k) to z`. 于是 `T z = lim T x_(n_k) = 0`, 即 `tilde z = tilde 0`. 因此 `||tilde x_(n_k)||` `= ||tilde x_(n_k) - tilde z||` `le ||x_(n_k) - z||` `to 0`, 与 `||tilde x_(n_k)|| = 1` 矛盾.

Fredholm 结论二 若 `T in cc L(X)`, 则 `bar("Im"T) = ("Ker"T^ast)^_|_`, `bar("Im"T^ast) = {::}^_|_"Ker"T`.
当 `A = I-T` 为紧算子时, 利用结论一, 等式中的闭包运算可以去掉.

只证第一个等式. 首先易证 `{::}^_|_"Im"T sube "Ker" T^ast`, `quad bar("Im"T) sube ("Ker"T^ast)^_|_`. 又 `N_1 sube N_2` `rArr N_2^_|_ sube N_1^_|_`, 因此 `bar("Im"T)` `sube ("Ker"T^ast)^_|_` `sube ({::}^_|_"Im"T)^_|_`. 接下来证明上式右边 `sube` 左边. `AA x_0 in ({::}^_|_"Im"T)^_|_` 有 `(AA f in {::}^_|_"Im"T)` `quad f(x_0) = 0`. 即 `(AA f in X^ast)` `quad f("Im"T) = 0 rArr f(x_0) = 0`. 根据 Hahn-Banach 定理的推论 (点与线性子空间的分离) 知道 `x_0 in bar("Im"T)`.

Fredholm 结论三 若 `A = fr C(X)`, `T = I - A`, 则 `"Ker"T = 0 rArr "Im"T = X`.

反设 `"Im"T != X`, 构造一系列子空间 `X_0 = X`, `X_(n+1) = T(X_n)`, 于是利用 `X_0 != X_1` 和 `T` 是单射得到 `X_0 supne X_1 supne X_2 cdots`. 又由结论一, `T` 是闭值域算子, 所以上式的子空间都是闭的. 根据 Riesz 引理, 赋范线性空间单位球面上的点到真闭子空间的距离可以取到无限接近 1, 因此 `AA n in NN`, `EE x_n in X_n`, `||x_n|| = 1`, 且 `d(x_n, X_(n+1)) ge 1//2`. 从而对 `AA m, n in NN`, `m lt n` 有 `||A x_m - A x_n||` `= ||x_m - T x_m + T x_n - x_n||` `ge 1//2`, `quad` 因为 `T x_m, T x_n, x_n in X_(m+1)`. 这与 `A` 的紧性矛盾.
从下式也可以看出每个子空间 `X_n` 都是闭的: `T^n = I + sum_(k=1)^n (n;k) (-A)^k` `= I + `紧算子.

结论四是关于维数的等式, 我们需要下面的同构引理:

设 `M sube X` 是闭线性子空间, 则 `(X//M)^ast cong {::}^_|_M` (等距线性同构).

Fredholm 结论四 若 `A = fr C(X)`, `T = I - A`, 则 `dim "Ker"T` `= dim "Ker"T^ast` `= op(codim)"Im"T` `= op(codim)"Im"T^ast` `lt oo`.

  1. `dim "Ker"T lt oo`: 注意到 `I|_("Ker"T)` `= A|_("Ker"T)` 是紧算子, 🚧
  2. 下证 `op(codim) "Im"T le dim "Ker"T`. 如若不然, `dim"Ker"T lt dim(X//"Im"T)`, 则存在 `X//"Im"T` 的真子空间 `K` 使得 `dim K = dim "Ker"T lt oo`, 故存在等距线性同构 `V: "Ker"T to K`. 记 `tilde T` 是 `T` 在商空间 `X//"Ker"T` 诱导的算子, 然后令 `T_1 := tilde T o+ V: X to "Im"T o+ K`. 于是 `"Ker" T_1 = 0`. 又 `I - T_1` 是紧算子 (因为 `I - tilde T` 是紧算子, `I - V` 为有限维算子, 也是紧算子), 由结论三得到 `X = "Im" T_1` `= "Im"T o+ K`, 这与 `K` 是 `X//"Im"T` 的真子空间矛盾.
  3. 综上有 `op(codim) "Im"T` `= dim(X//"Im"T)` `==^(??) dim (X//"Im"T)^ast` `= dim {::}^_|_"Im"T` `= dim {::}^_|_(("Ker"T^ast)^_|_)` `ge dim"Ker"T^ast`. 同理 `op(codim) "Im"T^ast` `ge dim"Ker"T^(ast ast)` `ge dim"Ker"T`, `quad` 因为 `T^(ast ast)` 是 `T` 的延拓. 结合以上两式, 利用 2. 的结论得到 `dim "Ker"T^ast` `le op(codim) "Im"T` `le dim"Ker"T` `le op(codim) "Im"T^ast`, 但 `I - T^ast` 也是紧算子, 由 2. 有 `op(codim)"Im"T^ast le dim"Ker"T^ast`, 所以上式只能全部取等.

紧算子的谱 (Riesz-Schauder 理论)

有穷维空间的紧算子就是线性代数中的有限维线性变换. 本节主要研究无穷维情形:

设 `A` 为有界线性算子, 则 `A` 的不同本征值对应的非零特征向量线性无关.

对特征向量的个数 `n` 做归纳. `n = 1` 时显然成立. 现在假设 `lambda_1, cdots, lambda_(n+1)` 是 `n+1` 个不同的本征值, 对应的特征向量是 `x_1, cdots x_(n+1)`. 若 `x_(n+1) = sum_(i=1)^n a_i x_i`, 则 `lambda_(n+1) x_(n+1) = A x_(n+1)` `= sum_(i=1)^n a_i lambda_i x_i`,
`lambda_(n+1) x_(n+1) = sum_(i=1)^n a_i lambda_(n+1) x_i`. 两式相减得 `sum_(i=1)^n a_i (lambda_i - lambda_(n+1)) x_i = 0`. 由归纳假设 `x_1, cdots, x_n` 线性无关, 故 `a_1 = cdots = a_n = 0`, 与 `x_(n+1) != 0` 矛盾. 因此 `x_1, cdots, x_(n+1)` 线性无关.

[知乎@PinkRabbit] 设 `S sube RR^n`, 若 `bar S` 不可数, 则 `S` 有不可数个聚点.
一个反直觉例子: 存在全由孤立点构成的可数集有不可数个聚点. 考虑构造 Cantor 集时每次删去的开区间, 在每个开区间中任选一点构成集合. 集合中的点全是孤立点, 但 Cantor 集 (不可数) 中的点都是它的聚点.

`bar S` 是 `S` 的聚点集合与孤立点集合的无交并. 但 `S` 的孤立点只有至多可数个, 这是因为每个孤立点都存在一个开邻域只含自己, 不含 `S` 的其它点. 但是 `RR^n` 上的不相交开集只有至多可数个 (每个开集中含有一个不同的有理点).

    无穷维紧算子的谱 设 `dim X = oo`, `A in fr C(X)`, 则
  1. `0 in sigma(A)`;
  2. 非零谱点一定是本征值: 若 `0 != lambda in sigma(A)`, 则 `lambda in sigma_p(A)`;
  3. `sigma(A)` 要么没有聚点, 要么以 `0` 为唯一聚点;
  4. `sigma(A)` 是至多可数集.
  1. 我们证明: 无穷维紧算子 `A` 没有连续逆. 参考无穷维空间单位球面不列紧的证明, 存在单位球面上的一个点列 `{x_n}`, 其任意两项的距离大于 `1//2`. 该点列在紧算子 `A` 下的像 `{A x_n}` 有收敛子列 `{A x_(n_k)}`. 假如 `A` 有连续的逆, 相应的子列 `{x_(n_k)}` 也应当收敛, 这是不可能的.
  2. 因为 `lambda != 0`, 由 Fredholm 结论三, `lambda I - A` 是单射等价于它是双射. 因此 `lambda` 不属于连续谱, 也不属于剩余谱.
  3. 反设 `sigma(A)` 有聚点 `lambda != 0`, 则存在 `sigma(A)` 中的数列 `lambda_n to lambda`, 其中 `lambda_n` 的项两两不同且不等于 `0`. 由 2. 知 `lambda_n` 的每一项都是本征值, 于是可以选出非零特征向量 `x_n in "Ker"(lambda_n I - A)`. 由引理知 `{x_n}` 的任意有限项线性无关. 令 `E_n := "span"{x_1, cdots, x_n}`, 则 `E_n subne E_(n+1)`, 由 Riesz 引理, `EE y_(n+1) in E_(n+1)`, `quad ||y_(n+1)|| = 1`, `quad d(y_(n+1), E_n) ge 1//2`. 注意 `lambda_n to lambda != 0`, 所以 `{lambda_n^-1 y_n}` 是有界点列. 另一方面, 对任意正整数 `m` 有 `lambda_m^-1 A y_m - y_m` `= lambda_m^-1 (A - lambda_m I) y_m in E_(m-1)`. 故 `m gt n` 时 `||lambda_m^-1 A y_m - lambda_n^-1 A y_n||` `= ||y_m - y_m + lambda_m^-1 A y_m - lambda_n^-1 A y_n||` `ge 1//2`. 因此 `{lambda_n^-1 A y_n}` 没有收敛子列, 这与 `A` 的紧性矛盾.
  4. 根据引理, 和 `RR^n` 一样, `CC` 上的不可数集有无穷多个聚点. 但 `sigma(A)` 只有至多一个聚点, 因此 `sigma(A)` 不能是不可数集.
    定理表明无穷维紧算子 `A` 的谱只有三种可能情形:
  1. `{0}`;
  2. `{0, lambda_1, cdots, lambda_n}`;
  3. `{0, lambda_1, lambda_2, cdots}`, 其中 `lambda_n to 0`.
    不变子空间 设 `A in cc L(X)`, 称 `M sube X` 是 `A` 的不变子空间, 如果 `A(M) sube M`.
  1. `{0}, X` 是平凡的不变子空间;
  2. 若 `M` 是不变子空间, 则 `bar M` 也是;
  3. 若 `lambda` 是 `A` 的本征值, 则特征子空间 `"Ker"(lambda I - A)` 是 `A` 的不变子空间. 特别当 `lambda != 0` 时, 由 Fredholm 结论, 它是有限维不变子空间 (从而闭);
  4. `AA y in X`, `P_y := {P(A) y | P "为多项式"}` 是 `A` 的不变子空间.

无穷维紧算子存在非平凡的闭不变子空间.

    设 `A in fr C(X)`.
  1. 若 `A = 0`, 则每个有限维子空间都是闭不变子空间. 若 `A != 0`, 且存在本征值 `lambda != 0`, 由引理知 `"Ker"(lambda I - A)` 是闭不变子空间. 它不等于 `X`, 否则推出 `A = lambda I`, 这不是紧算子.
  2. 现在设 `A != 0`, 且只有零本征值. 于是 `sigma(A) = {0}`. 从引理知道 `AA y in Y - {0}`, `P_y` 是 `A` 的不变子空间. 使用反证法, 设 `A` 只有平凡的闭不变子空间, 则 `bar(P_y) = X`.
  3. 不妨设 `||A|| = 1`, 取 `x_0 in X` 使 `||A x_0|| gt 1`, 于是 `||x_0|| = ||A|| ||x_0|| gt 1`. 令 `C := bar(A(B(x_0, 1)))`, 则 `C` 是紧集, 且 `0 !in C`. `AA y in C`, 则 `y != 0`. 由 `bar(P_y) = X` 知道, `EE A` 的多项式 `T_y = P(A)` 使得 `||T_y y - x_0|| lt 1`. 由范数的连续性, `EE delta_y gt 0`, 使得 `||T_y z - x_0|| lt 1`, `quad AA z in B(y, delta_y)`. 由 `C` 的紧性, 存在有限子覆盖 `C sube uuu_(i=1)^n B(y_i, delta_i)`. 故 `AA y in C`, `EE i` 使得 `||T_(y_i) y - x_0|| lt 1`, 即 `T_(y_i) y in B(x_0, 1)`.
  4. 简记 `T_i := T_(y_i)`, 则由 `T_i y in B(x_0, 1)` 知道 `T_i A y = A T_i y in C`. 于是又 `EE j` 使得 `T_j T_i A y in B(x_0, 1)`. 于是对任意正整数 `k`, 都存在 `i_1, cdots, i_(k+1)` 使得 `||T_(i_1) cdots T_(i_(k+1)) A^k y - x_0|| lt 1`. 这推出 `||x_0|| - 1 lt ||T_(i_1) cdots T_(i_(k+1)) A^k y||`. 令 `mu := max_(1 le i le n) ||T_i|| gt 0`, 则 `||x_0||-1 le mu^(k+1) ||A^k|| ||y||`, `quad AA y in C`, `AA k in NN`.
  5. `A` 的谱半径为 0, 根据 Gelfand 定理, `lim_(k to oo) ||A^k||^(1//k) = 0`. 另一方面由 `y != 0` 我们有 `||A^k||^(1//k)` `ge 1/mu ((||x_0||-1)/(mu||y||))^(1//k)` `to 1/mu` `(k to oo)`, 从而矛盾.

零链长与像链长 设 `T in cc L(X)`, 则有 `{0} sube "Ker"T sube "Ker"T^2 sube cdots`, 若存在最小非负整数 `p` 使 `"Ker"T^p = "Ker"T^(p+1)`, 则对任意 `n ge p` 都有 `"Ker"T^p = "Ker"T^n`. 称 `p` 为 `T` 的零链长, 记为 `p = p(T)`. 同理 `X supe "Im"T supe "Im"T^2 supe cdots`, 若存在最小非负整数 `q` 使 `"Im"T^q = "Im"T^(q+1)`, 则对任意 `n ge q` 都有 `"Im"T^q = "Im"T^n`. 称 `q` 为 `T` 的像链长, 记为 `q = q(T)`.
特别, `p = 0` `iff "Ker"T = {0}` `iff T` 是单射,
`q = 0` `iff "Im"T = X` `iff T` 是满射.

  1. 设 `"Ker"T^p = "Ker"T^(p+1)`, 任取 `x in "Ker"T^(p+2)`, 则 `T x in "Ker"T^(p+1) = "Ker"T^p`, 于是 `x in "Ker"T^(p+1)`.
  2. 设 `"Im"T^q = "Im"T^(q+1)`, 任取 `y = T^(q+1) x in "Im"T^(q+1)`, 则 `T^q x in "Im"T^q = "Im"T^(q+1)`, 于是 `y = T(T^q x) in "Im"T^(q+2)`.

设 `T = I - A`, `A in fr C(X)`, 则 `T` 的零链长与像链长都有限且相等: `p(T) = q(T) lt oo`.

  1. `q lt oo`: 如若不然, 则有 `"Im"T supne "Im"T^2 supne cdots`, 从 Fredholm 结论三的证明知道这是不可能的.
  2. `p le q`: 由定义 `"Im"T^q = "Im"T^(q+1)`. 由于 `T = I +`紧算子, 所以 `T^q = I +`紧算子. 应用 Fredholm 结论四, `dim "Ker"T^q` `= "codim" "Im"T^q` `= "codim" "Im"T^(q+1)` `= dim"Ker"T^(q+1) lt oo`. 这推出 `"Ker"T^q = "Ker"T^(q+1)`, 于是 `p le q`.
  3. `q le p`: 类似由定义 `"Ker"T^p = "Ker"T^(p+1)` 和 Fredholm 结论四, 有等式 `"codim" "Im"T^(p+1)` `= dim "Ker"T^(p+1)` `= dim "Ker"T^p` `= "codim""Im"T^p lt oo`.

设 `T = I - A`, `A in fr C(X)`, 则有直和分解 `X = "Ker"T^p o+ "Im"T^p`, `quad p = p(T)`, 且 `T_1 := T|_("Im"T^p)` 存在连续逆算子.

  1. `X = "Ker"T^p + "Im"T^p`: 任取 `x in X`, 由于零链长等于像链长, 有 `T^p x in "Im"T^p = "Im"T^(2p)`, 即存在 `y in X` 使得 `T^p x = T^(2p) y`. 从而有分解 `x = x - T^p y + T^p y`, 其中 `T^p y in "Im"T^p`, 而 `x - T^p y in "Ker"T^p`.
  2. `"Ker"T^p nn "Im"T^p = {0}`: 任取 `T^p x in "Im"T^p`, 若 `T^p x in "Ker"T^p`, 则 `x in "Ker"T^(2p) = "Ker"T^p`. 于是 `T^p x = 0`.
  3. `T_1` 存在连续逆. 由逆算子定理, 只需证 `T_1` 是双射. 显然 `T("Im"T^p) = "Im"T^(p+1) = "Im"T^p`, 所以 `T_1` 是满射. 下证 `T_1` 是单射, 这是因为 `"Ker"T_1 = "Ker"T nn "Im"T^p` `sube "Ker"T^p nn "Im"T^p` `= {0}`.

设紧算子 `A` 的非零本征值为 `{lambda_i}_(i=1)^oo`, 令 `T_i = lambda_i I - A`, `p_i = p(T_i)`, 则 `A` 在子空间 `O+_(i=1)^oo "Ker" T_i^(p_i)` 上有 Jordan 标准形. 其关键在于, 每个 `V_i := "Ker"T_i^(p_i)` 是 `A` 的有限维不变子空间, 且 `T_i` 限制在 `V_i` 上是幂零算子. 根据线性代数, 存在 `V_i` 的一组基, 使得 `T_i|_(V_i)` 的表示矩阵是若干 Jordan 块, 其对角线为零. 于是 `A|_(V_i)` 的表示矩阵也是若干 Jordan 块, 其对角线为 `lambda_i`.

Hilbert-Schmidt 定理

本节的目标是将线性代数中的谱定理 (实对称阵与 Hermite 阵的对角化) 推广到 Hilbert 空间. 如无说明, 本节的 `H` 均表示 Hilbert 空间.

称 `A in cc L(H)` 为对称算子, 如果 `(A x, y) = (x, A y)`, `quad AA x, y in H`. 可以看出 `A` 是一个自伴算子, 即 `A = A^ast`.

`RR^n` 中的实对称矩阵是对称算子; `CC^n` 中的 Hermite 阵 (`A = bar(A^(sf T))`) 是对称算子.

自伴算子 (自共轭算子) `A = A^ast`, 反伴随算子 `A = -A^ast`, 正规算子 `A A^ast = A^ast A`, 酉算子 `T^-1 = T^ast`.

    `H` 上对称算子的性质
  1. `A` 对称当且仅当 `AA x in H`, `(A x, x) in RR`.
  2. 对称算子之和是对称的; 对称算子乘以实数是对称的; 对称算子在子空间上也是对称的.
  3. 设 `A` 对称, 则 `sigma(A) sube RR`, 并且 `||(lambda I - A)^-1|| le 1/|"Im" lambda|`, `quad AA lambda in CC - RR`.
  4. 设 `A` 对称, 则不同本征值对应的特征向量相互正交.
  5. 设 `A` 对称, 则 `Sup_(||x||=1) |(A x, x)| = ||A||`.
  1. 记 `a(x, y) = (A x, y)`, 则 `a` 是共轭双线性函数. 于是 `A` 对称
    `iff (AA x, y in H)` `(A x, y) = (x, A y)`
    `iff (AA x, y in H)` `a(x, y) = bar(a(y, x))`
    `iff a` 共轭对称     根据第四章共轭对称性的等价条件知道, 这等价于 `AA x in H`, `a(x, x) = (A x, x) in RR`.
  2. 由 1. 立即得到.
  3. 我们来证 `CC - RR` 中都是正则点. 首先 `AA lambda = u + "i"v in CC`, `u, v in RR`, 由 2. 知 `B := u I - A` 对称, 所以 `AA x in H`, `((u I - A) x, "i"v x) + ("i"v x, (u I - A) x)` `= -"i"v(B x, x) + "i"v (x, B, x)` `= 0`. 这推出 `AA lambda in C`, `AA x in H`, `||(lambda I - A) x||^2 = ||(u I - A)x||^2 + |v|^2 ||x||^2` `ge |v| ||x||`. (`ast`) 故 `v != 0` 时逆算子存在且 `||(lambda I - A)^-1|| le |v|^-1`. 下设 `lambda in CC - RR`, 要证 `"Im"(lambda I - A) = H`. 事实上, `AA x, y in H`, `((lambda I - A) x, y)` `= (x, (lambda I - A)^ast y)` `= (x, (bar lambda I - A^ast) y)` `= (x, (bar lambda I - A) y)`. 因此 `"Im"(lambda I - A)^_|_` `= "Ker"(bar lambda I - A)` `==^"由 (*)" {0}`.
  4. 设 `lambda, mu in RR` 是两个本征值, `x in "Ker"(lambda I - A)`, `y in "Ker"(mu I - A)`, 则 `lambda (x, y)` `= (A x, y)` `= (x, A y)` `= mu (x, y)`. 若 `lambda != mu`, 则 `(x, y) = 0`.
  5. 记 `c := Sup_(||x||=1) |(A x, x)|`. 由第四章有界共轭双线性函数的内积表示, `||A|| = Sup_(||x|| = ||y|| = 1) |(A x, y)|`. 显然有 `c le ||A||`. 另一方面, 由对称性 `(A(x+y), x+y) - (A(x-y), x-y)` `= 2(A x,y) + 2(A y,x)` `= 4 "Re"(A x, y)`. 且 上式左边 `le c(||x+y||^2 + ||x-y||^2)` `==^"极化恒等式" 2c(||x||^2 + ||y||^2)`. 这推出 `"Re"(A x, y) le c`, `quad AA ||x|| = ||y|| = 1`. 对任意给定的 `||x|| = ||y|| = 1`, 可以选取单位复数 `|a| = 1` 使得 `|(A x, y)|` `= a(A x, y)` `= (A x, bar a y)` `= "Re"(A x, bar a y)` `le c`. 因此 `||A|| le c`.

设 `A in cc L(H)` 对称, 考虑单位球面上的二次型 `(A x, x)`, `||x|| = 1`. 我们知道 `(A x, x)` 是实数, 且它的绝对值以 `||A||` 为上确界. 现在设 `A` 是紧算子, 则该上确界可以取到, 这就是:

对称紧算子的极值性质 设 `A in fr C(H)` 对称, 则 `EE ||x_0|| = 1`, 使得 `|(A x_0, x_0)| = max_(||x|| = 1) |(A x, x)| = ||A||`. 如果记 `lambda = (A x_0, x_0)`, 我们还有 `A x_0 = lambda x_0`. 即 `lambda` 是本征值, `x_0` 是相应的特征向量.

  1. 不妨设 `Sup_(||x||=1) |(A x, x)| = Sup_(||x||=1) (A x, x) := lambda`, 否则可以用 `-A` 代替 `A`. 在单位球面上取点列 `{x_n}` 使 `(A x_n, x_n) to lambda`. 注意 Hilbert 空间是自反空间, 而自反空间的单位闭球弱自列紧, 可知 `{x_n}` 有弱收敛的子列, 仍记作 `{x_n}`. 并记 `x_n ⇀ x_0`, 于是 `||x_0|| le 1`. 利用 `A` 的紧性, `A x_n to A x_0`, 从而 `(A x_n, x_n) to (A x_0, x_0)`. 于是由极限的唯一性知 `lambda = (A x_0, x_0)`.
  2. 下证 `||x_0|| = 1`. 事实上若 `||x_0|| lt 1`, 则 `(A x_0, x_0)` `le ||A|| ||x_0||^2` `lt ||A|| = lambda`, 矛盾.
  3. 下证 `A x_0 = lambda x_0`. `AA y in H`, 考虑定义在复平面原点邻域内的函数 `varphi_y(t) := ((A(x_0 + t y), x_0 + t y))/((x_0 + t y, x_0 + t y))`. 由 `x_0` 的定义, 函数 `varphi_y` 在 `t = 0` 处达到极大, 于是 `varphi_y'(0) = 0`, 即 `"Re"(y, A x_0 - lambda x_0) = 0`. 由 `y` 的任意性推出 `A x_0 = lambda x_0`.

谱定理 (Hilbert-Schmidt) 设 `A in fr C(H)` 对称, 设 `A` 的本征值为 `{lambda_i}`, 则存在一组正交规范基 `{e_i}`, 满足 `AA x in H`, `x = sum (x, e_i) e_i`,
`A x = sum lambda_i (x, e_i) e_i` `= sum lambda_i (e_i ox e_i) x`. (`ast`) 换言之, 对称紧算子 `A` 可以对角化.

  1. 任取非零本征值 `lambda`, 取 `"Ker"(lambda I - A)` 的正交规范基 `{e_i^((lambda))}_(i=1)^(m(lambda))`, `quad m(lambda) = dim "Ker"(lambda I - A) lt oo`. 若 `0` 是本征值, 则取 `"Ker"A` 的正交规范基 `{e_i^((0))}`, 它不一定可数 (`H` 未必是可分空间). 现在令 `{e_i'} := {e_i^((lambda)): lambda "是非零本征值", i = 1, cdots, m(lambda)}`,
    `{e_i} := { {e_i'}, if 0 "不是本征值"; {e_i'} uu {e_i^((0))}, otherwise :}`     由于每个 `e_i` 都是 `A` 的特征向量, 故在 `M := "span"{e_i}` 上等式 (`ast`) 成立.
  2. 下证 `M^_|_ = {0}`. 如若不然, 对 `AA x in M^_|_, AA e_i`, 由对称性有 `(A x, e_i)` `= (x, A e_i)` `= (x, lambda_i e_i)` `= 0`. 即 `A x in M^_|_`, `M^_|_` 是 `A` 的不变子空间. 记 `tilde A := A|_(M^_|_)`, 则由前一定理, 存在 `tilde A` 的单位特征向量 `x_0 in M^_|_`, `||x_0|| = 1`. 另一方面, 显然 `x_0` 也是 `A` 的特征向量, 所以 `x_0 in M`, 矛盾.
    3. 关于求和的可数性: 从证明过程看出, 虽然基底 `{e_i}` 可能是不可数的, 但只有至多可数个 `e_i` 对应的本征值 `lambda_i != 0`. 此外由 Bessel 不等式知道 Fourier 系数 `(x, e_i)` 也只有至多可数个不为零.
