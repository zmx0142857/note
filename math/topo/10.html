Jones 多项式

    Jones 多项式 对任意链环的有向投影图 `L`, 将它对应为变元 `s` 的一元函数 `V(L)(s)`, 满足下面性质:
  1. 同痕不变性 若有向投影图 `L`, `L'` 同痕, 则 `V(L) = V(L')`.
  2. 标准值 平凡纽结 "〇" 对应的函数为 `V(〇) = 1`.
  3. 拆接关系式 `s^2 * V(左手) - s^-2 * V(右手) + (s - s^-1) * V(双手) = 0`. 其中左手 ()、 右手 ()、 双手 () 代表三个几乎一样的有向投影图, 只在某个交叉点附近的连接方式不同.

    把这三个图形想像成面对自己的一个人🙅‍♀️, 左手图形代表这个人交叉双臂, 左臂在前; 右手也作类似的解释.

    4. 最后, 把 `s` 换成 `t^(1//2)`, 得到的关于 `t` 的函数 `J(L)(t)` 称为 Jones 多项式. 这个 "多项式" 允许 `t` 的次数是负整数和半整数, 且系数全为整数. 拆接关系式是 Jones 多项式的一切精华.
    Jones 多项式的缩写 按各项次数从小到大连续写出系数:
  1. 如果各项次数是整数, 就在常数项加一个下划线; 比如 `t^-1 -3 - t^2` 缩写为 `1 - ul 3 + 0 - 1`.
  2. 如果各项次数是半整数, 就用斜杠把正负次数的项分开; 比如 `2 t^(-3//2) - t^(1//2)` 缩写为 `2 + 0 // -1`.
  1. 设 `L, L'` 互为镜像, 则 `J(L')(t) = J(L)(t^-1)`, 换言之次数全部添负号. 8 字结不具有手性, 其 Jones 多项式 `1 -1 + ul 1 -1 +1` 是完全对称的.

计算 `5_1` (左手星星结) 和 `5_2` (左手88结) 的 Jones 多项式. 

from sympy import *
s = S('s')
A = s**2
B = -s**-2
C = s-s**-1
L_5_1 = (B*(-s**-8+s**-6+s**-2)+C*(-s**-11+s**-9-s**-7-s**-3))/(-A)
expand(L_5_1) # 左手星星结 s**(-4) + s**(-8) - 1/s**10 + s**(-12) - 1/s**14
L_5_2 = (B*(-s**-8+s**-6+s**-2)+C*(-s**-5-s**-1))/(-A)
expand(L_5_2) # 左手88结 s**(-2) - 1/s**4 + 2/s**6 - 1/s**8 + s**(-10) - 1/s**12
