# GUI 图形界面

## 桌面端

- c#: wpf, winforms, uwp (xaml), maui (xaml), gtk#, avalonia
- cpp: qt (qml), gtk (多语言、跨平台), fltk (轻量但不好看), mfc (已过时), sfml (过于底层), win32api (过于底层)
- java: javafx (fxml), swing (已过时), awk (已过时)
- python: pyqt (对 qt 的封装), PySide (还是对 qt 的封装), tkinter (对 tk 的封装)
- js: electron, tauri
- rust: egui, gpui

## 移动端

- kotlin: android 原生
- swift: ios 原生
- dart: flutter
- js: ReactNative, uni-app
- lua: AndroLua (玩具)

## 游戏

- cpp: unreal
- c#: unity, godot
- java: libgdx
- js: cocos
- lua: love

## 底层图形接口

- 跨平台: opengl, vulkan
- macOS: metal
- windows: directX
- web 端: webgl, webgpu
