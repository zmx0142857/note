1. 根据定义证明: 对任意 `a gt 1` 都有 `lim_(n to oo) n/(log_a n) = +oo`.
  2. 证明:
    1. `lim_(n to oo) (1 + 1/3 + 1/5 + cdots + 1/(2n-1)) = +oo`;
    2. `lim_(n to oo) (1/(a+b) + 1/(2a+b) + 1/(3a+b) + cdots + 1/(na+b) = +oo`, 其中, `a gt 0`, `b gt 0`;
    3. `lim_(n to oo) (1/root(4)(1^3*2) + 1/root(4)(2^3*3) + 1/root(4)(3^3*4) + cdots` `+ 1/root(4)(n^3*(n+1))) = +oo`.
  3. 证明: 当 `n to oo` 时,
    `1 + 1/3 + 1/5 + cdots + 1/(2n-1) ~ 1/2 + 1/4 + cdots + 1/(2n)`.
  4. 证明:
    1. 如果 `x_n ~ y_n`, `y_n ~ z_n`, 则 `x_n ~ z_n`;
    2. 如果 `x_n ~ y_n`, 则 `x_n^m ~ y_n^m`, `root(m)(x_n) ~ root(m)(y_n)` (`m in NN`; 假设开方运算有意义);
    3. 如果 `x_n ~ u_n`, `y_n ~ v_n`, 则 `x_n y_n ~ u_n v_n`.
  5. 设 `{x_n}` 是无穷小量. 证明:
    1. 对多项式 `P(x) = a_0 x^m + a_1 x^(m+1) + cdots + a_k x^(m+k)`, 其中 `a_0 != 0`, 有 `P(x_n) ~ a_0 x_n^m`;
    2. `sqrt(x_n + sqrt(x_n + sqrt(x_n))) ~ root(8)(x_n)` (`x_n gt 0`);
    3. `(1+x_n)^m = 1 + mx_n + o(x_n)` (m 是正整数).
  6. 设 `{x_n}` 是无穷大量. 证明:
    1. 对多项式 `P(x) = a_0 x^m + a_1 x^(m-1) + cdots + a_(m-1) x + a_m`, 其中 `a_0 != 0`, 有 `P(x_n) ~ a_0 x_n^m`;
    2. `sqrt(x_n + sqrt(x_n + sqrt(x_n))) ~ sqrt(x_n)` (`x_n gt 0`);
    3. `(1+x_n)^m ~ x_n^m` (`m gt 0`).
