1. 设函数 `f` 在区间 `I` 上满足以下条件: 存在常数 `mu gt 1` 和 `C gt 0` 使成立 `|f(x) - f(y)| le C|x-y|^mu`, `AA x, y in I`. 证明: `f` 在区间 `I` 上是常值函数.
  2. 设函数 `f` 和 `g` 在区间 `I` 上可微, `g(x) != 0`, `AA x in I`, 且成立 `| (f(x), g(x)), (f'(x), g'(x)) | = 0`, `AA x in I`. 证明: 存在常数 `c` 使 `f(x) = cg(x)`, `AA x in I`, 并举例说明, 为保证这个结论成立, 条件 "`g(x)` 没有零点" 不能去掉.
  3. 设函数 `f` 在区间 `I` 上可微且导数是一常数: `f'(x) = a`, `AA x in I`. 证明: `f` 在区间 `I` 上是一线性函数 `f(x) = ax+b`, `AA x in I`, 其中 `a`, `b` 是常数.
  4. 设 `I` 是正半轴上的一个区间, 函数 `f` 在 `I` 上可微且满足 `xf'(x) + af(x) = 0`, `AA x in I`, 其中 `a` 是常数. 证明: 存在常数 `C` 使成立 `f(x) = Cx^-a`, `AA x in I`.
  5. 设函数 `f` 在区间 `I` 上可微. 证明:
    1. 如果 `f'(x) = be^(ax)`, `AA x in I`, 其中 `a`, `b` 是常数, 则 `f(x) = b/a e^(ax) + c`, `AA x in I`, 其中 `c` 是常数;
    2. 如果成立 `f'(x) = af(x) + b`, `AA x in I`, 其中 `a`, `b` 是常数, 则 `f(x) = c e^(ax) - b/a`, `AA x in I`, 其中 `c` 是常数.
  6. 应用导数证明下列恒等式:
    1. `arcsin x + arccos x = pi/2`;
    2. `arctan x = arcsin {:x/sqrt(1+x^2):}`;
    3. `arccos{:1/sqrt(1+x^2):} + arccos{:x/sqrt(1+x^2):}` `= { pi/2, if x ge 0; 2 "arccot" x - pi/2, if x lt 0; :}`;
    4. `2arctan x + arcsin{:(2x)/(1+x^2):} = pi sgn x` (`|x| ge 1`).
  7. 设函数 `f` 在 `[0, +oo)` 上连续, 在 `(0, +oo)` 上可微, 且存在常数 `C gt 0` 使得 `|f'(x)| le C|f(x)|`, `AA x gt 0`. 又设 `f(0) = 0`. 证明: `f(x) = 0`, `AA x ge 0`.
  8. 设函数 `f` 在区间 `I` 上可微, 且存在常数 `a gt 0` 使得 `|f'(x)| le a`, `AA x in I`. 证明: 存在常数 `b gt 0` 使 `|f(x)| le a|x| + b`, `AA x in I`.
